A simple base web application scaffold for a `TypeScript + React + Vite` web app:

```shell
npm create vite@latest my-app-name -- --template react-ts
```

Follow the prompts, then change into the `my-app-name` directory and install:

```shell
cd my-app-name
npm i
```
