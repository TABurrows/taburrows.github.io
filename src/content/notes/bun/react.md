How to build a React app with the Bun runtime.

## Quickstart

Create a React app skeleton with Bun:

```
bun init --react
```

Run the React app in Dev mode:

```
bun dev
```

Build a static site for production:

```
bun run build
```

Run the server in production:

```
bun start
```
