## Hyperledger

A Linux Foundation Project consisting of a number of open source blockchain tools for the building, testing, deployment and management of private blockchain networks.

### Hypderledger Fabric

"an enterprise-grade permissioned distributed ledger framework for developing solutions and applications"

### Hyperledger Caliper

"a blockchain performance benchmark framework, which allows users to test different blockchain solutions with predefined use cases"

### Hyperledger Composer

"composer uses LoopBack to connect your blockchain to your existing systems"

### Ganache

"quickly fire up a personal Ethereum blockchain which you can use to run tests, execute commands, and inspect state while controlling how the chain operates"

### Truffle Suite

"interact directly with the blockchain using Truffle console. Write tests in Solidity, JavaScript, and TypeScript"
