Networks - Network Tags


Networks use network tags to identify which VM instances are subject to certain firewall rules and network routes. Later in this lab, you create a firewall rule to allow HTTP access for VM instances with the web-server tag. 

example, if you check the Allow HTTP traffic checkbox, which would tag an instance as http-server and create the tagged firewall rule for tcp:80 for you


The default-allow-internal firewall rule allows traffic on all protocols/ports within the default network.
